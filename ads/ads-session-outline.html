***David Meeting Ad Info Notes***


Basic vocabulary

Ad types

  • Linear ads: Ads that occur during playback, take up time in the timeline (example is a pre-roll)
  • Non-linear ads: Does not interrupt playback; overlay and companion ads are examples

Ad standards

  • VMAP: "ad playlist"; specifies pre-roll, mid-roll(s) and post-roll (URL to a VMAP document); format is XML

    • DFP calls VMAP documents "ad rules"; with ad rules you can configure all your ads on the server and any client who understands ad rules behaves the same; in "bad old days" ad configuration at the client; logic moved into publishing system

  • VAST: Individual ad details; VMAP points to a VAST document with information about the ad

  • Example VAST from Network tab in Chrome Dev Tools

  • VPAID: Like VAST, but a standard for interactive ads; ad contains a quiz or goofy game (no BC Player support yet)

IMA Plugin information

  • IMA constraint: 2 versions of IMA library, Flash or HTML version, cannot use both

    • default settings for plugin use Flash if available (because Flash ads are so prevalent)
    • In config options for plugin can switch default to HTML (adTechOrder)
    • Desktop default Flash, mobile default HTML

  • Basic implementation of IMA Plugin

    • Include JavaScript for plugin
    • Add IMA Plugin to player
    • Provide serverURL to ad tag
    • Sample config

  • Dynamic Ad Targeting: Configuration of IMA Plugin done after gathering customer info (for example viewers age, then select ad rule)

    • Include JavaScript for plugin
    • Call IMA plugin and pass options at that point
      • myPlayer.ima3({ serverURL: ...})
  • Companion ads: IMA Plugin will deal with linear ads and overlays seamlessly, companion ads can be tricky since communication between the player and the rest of the page must occur; ; ; use In-page embed implementation
    • Use Google Publisher Tags (GPT) library and will automatically fill the companion ad blocks
    • Good tutorial available on IMA site
    • Use In-page embed implementation for companion ads since iframe can't interact with parent page
    • Example: JS and HTML for companion ad
