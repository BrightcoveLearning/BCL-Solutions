Analytics Report for Multiple Accounts

Note: the v1 version of the Analytics API makes generating reports very simple. Only a single api call using the parameter accounts=account1_id,account2_id,account3_id is all it takes.

In addition, the Analytics Report Builder will allow you to generate reports on multiple accounts. The Report Builder is currently in pre-release mode as part of the new Brightcove Studio - if you are interested in getting access to it, contact your account manager.

For these reasons, this sample has been retired.
