Get Brightcove API Client Credentials

You can use this app to get client credentials for Brightcove APIs. Before using this app, you should read the Brightcove OAuth Overview so that you understand what you are doing and how the client credentials are used in making API call. After you get your client credentials, you can use them with the Brightcove Learning Services proxies to test API calls.

Instead of using this app, you can also get your credentials via CURL or Postman - see:

  • Get Client Credentials Using CURL
  • Get Client Credentials Using Postman

Requirements

To use this app, you will need:

  • a Video Cloud or Perform account that you can login to

Note: you must be an administrator for the account to get client credentials. If you are not an administrator, talk to your account administrator about that.

Getting your BC_TOKEN

To use this app, you need to get your BC_TOKEN:

  1. Login to your Video Cloud or Perform account
  2. If you have multiple accounts, make sure you are in the one that you want to make API calls for
  3. Open the Developer JavaScript Console in your browser (see the browser help for how to do this if you don't know)
  4. Copy the following lines of JavaScript, paste them into your JavaScript console, and press Return: 
    var cookiesArray = document.cookie.split(";"), cookiesObj = {}, i, tmpArray = [];
for (i = 0; i < cookiesArray.length; i++) {
    tmpArray = cookiesArray[i].split("=");
    cookiesObj[tmpArray[0].replace(/\s/g, "")] = tmpArray[1];
}
window.prompt("BC_TOKEN:", cookiesObj.BC_TOKEN);
  5. You should see a prompt pop up that contains your BC_TOKEN - copy and paste it into the BC_TOKEN field below

Get your client credentials

Fill in the fields below and click the Get Client Credentials button to retrieve your client credentials. Copy them and save them, as you will Get Client Credentialsnot be able to retrieve them later (although you can always get new ones).
